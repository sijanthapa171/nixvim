# Welcome to Presenting.nvim

A slides presentation system for Neovim

Press `n` for next slide

# Features

- Create slides using Markdown
- Each slide starts with # heading
- Clean presentation mode
- Vim-style navigation

# Navigation Keys

- `n` - Next slide
- `p` - Previous slide
- `q` - Quit presentation

# How to Use

1. Create a markdown file with slides
2. Start each slide with # heading
3. Run `:Presenting` to start
4. Use navigation keys

# Code Examples

```lua
-- Example Lua code
local function hello()
  print("Hello from presentation!")
end
```

```nix
# Example Nix code
{ pkgs, ... }: {
  plugins.present.enable = true;
}
```
